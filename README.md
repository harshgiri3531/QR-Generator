# 📌 Overview

QRGen is a simple and modern **QR Code Generator Web Application** developed using **Flask**, **Python**, **HTML**, and **CSS**.

This application allows users to:

- 🔳 Generate QR Codes instantly
- 🌐 Convert any Text or URL into QR Code
- 🔐 Register & Login Users
- 🎨 Experience a clean modern UI
- ⚡ Generate QR codes quickly and easily

The project is beginner-friendly and perfect for learning:

- Flask Routing
- User Authentication UI
- Template Rendering
- Web Development with Python
- Frontend + Backend Integration

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/harshgiri3531/QR-Generator.git
```

---

## 2️⃣ Move into Project Folder

```bash
cd QR-Generator
```

---

## 3️⃣ Install Required Packages

```bash
pip install flask qrcode pillow
```

---

## 4️⃣ Run Flask Application

```bash
python app.py
```

---

# 🌐 Open in Browser

```bash
http://127.0.0.1:5000
```

# 🚀 How It Works

1. Open the application
2. Register/Login
3. Go to QR Generator page
4. Enter any text or URL
5. Click on **Generate QR Code**
6. QR code gets generated instantly
